# Проект API на базе FastAPI

## Описание проекта
FastAPI-приложение для управления благотворительными проектами и пожертвованиями. 
Система позволяет пользователям делать пожертвования в проекты, а администраторам — управлять проектами и получать аналитические отчеты.

## Основной функционал

### Аутентификация и авторизация
* Регистрация и авторизация пользователей JWT аутентификация с использованием FastAPI Users
* Ролевая модель: обычные пользователи и суперпользователи
* Защита эндпоинтов с использованием зависимостей FastAPI

### Пожертвования
* Создание пожертвований в открытые проекты
* Автоматическое распределение пожертвований по проектам
* Суммирование пожертвований по проектам

### Благотворительные проекты
* Создание, редактирование и удаление проектов
* Просмотр списка всех проектов
* Получение информации о конкретном проекте
* Автоматическое обновление статуса проекта при достижении цели

### Отчетность и аналитика
* Генерация отчетов в Google Таблицах
* Автоматическое создание таблицы с данными о проектах, отсортированными по скорости завершения
* Автоматическое заполнение таблицы актуальными данными из базы
* Настройка прав доступа к таблице через Google API
* Возврат ссылки на созданный отчет для удобного доступа

# Установка. 
## Следуйте следующим команда для установки и развертывание проекта у себя локально 
### Клонировать репозиторий и перейти в него в командной строке:
```
git clone https://github.com/Sava151/QRkot_spreadsheets.git
```
### Cоздать и активировать виртуальное окружение:
#### Рекомендуется использовать python 3.9
```
cd cat-charity-1/
```
```
py -3.9 -m venv venv
```
```
source venv/Scripts/activate
```
#### Уточнение имеющихся версий python 
```
py -0
```
#### Уточнение версии по умолчанию
```
python --version 
```
### Установить зависимости из файла requirements.txt
```
pip install -r requirements.txt
```
### Запустить проект
##Создайте файл .env:
```
DATABASE_URL=postgresql+asyncpg://user:password@localhost/dbname
SECRET=your_secret_key
```
```
 alembic upgrade head
```
```
 uvicorn app.main:app
```
